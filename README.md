# Healthcare_data_analysis_Project


## Project Overview
This project analyzes Healthcare patient encounter data analysis using SQL and Power BI to identify high-cost patients, financial risk factors, and cost patterns for better decision-making.

## Tools Used
- MS SQL Server
- Power BI
- CSV Datasets
  
## Key Analysis
- Evaluation of financial risk by encounter outcome  
- Identification of high-cost patients with repeated encounters  
- Analysis of total cost contribution by patient  
- Cost comparison across encounter types and payers  
- Trend analysis of healthcare expenses
